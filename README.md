# Convert PNG to ROM art
> Create ```.art``` crc32 format ROM art

## Prerequisites

```shell
# Homebrew
$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

# Node.js
$ brew install node
```

## Installation

```
# Navigate to app folder
$ cd app

# Install NPM packages
npm i

# run it
$ npm run dev
```
